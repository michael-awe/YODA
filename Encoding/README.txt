This folder contains everything needed to encode and image with a secret message and to decode the message using Python.

Contains:
1. babyyoda.png: The image that can be encoded
2. obiwan.wav: The secret audio message that will be encoded into the image
3. Encoding.py: The Python encoder 
4. GoldenMeasure.py: The Python decoder that allows for golden measure testing

How to use:
1. Download the project
2. Enter into the Encoding directory 
3. Run the Encoding.py
4. View the "Encoded_Image.png" that is produced
5. Run the GoldenMeasure.py
6. Play the "Decoded_Audio.wav" that is produced to listen to the sectret message
